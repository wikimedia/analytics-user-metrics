File Format

The command line utility (userstats) expects a CSV file with the following format 

username      project
Evan (WMF)    en
  • username: straightforward utf-8 encoded names or user ids.
  • project: the langauge or project code on which that username exists and up-to-date list of codes can be found here (the actual codes are in the fourth column of the tables, labeled "Wiki") .
